# AI для людей. Ідеї для бізнесу

Це репозиторій для вашого нового блогу на GitHub Pages, створеного на основі шаблону **Jekyll Clean Blog** від Start Bootstrap.

## Як опублікувати свій блог

Для публікації вашого блогу на GitHub Pages виконайте наступні кроки:

### 1. Створення репозиторію на GitHub

1.  Створіть новий репозиторій на GitHub.
2.  **Важливо:** Якщо ви хочете, щоб ваш блог був доступний за адресою `yourusername.github.io`, назвіть репозиторій **`yourusername.github.io`** (замініть `yourusername` на ваш нікнейм на GitHub).
3.  Якщо ви хочете, щоб блог був доступний за адресою `yourusername.github.io/blog-name`, назвіть репозиторій **`blog-name`**.

### 2. Завантаження файлів

1.  Ініціалізуйте Git у цій локальній папці:
    ```bash
    cd clean-blog-jekyll
    git init
    git branch -M main
    ```
2.  Додайте всі файли та зробіть перший коміт:
    ```bash
    git add .
    git commit -m "Initial commit: Setup Jekyll Clean Blog"
    ```
3.  Підключіть ваш віддалений репозиторій (замініть URL на ваш):
    ```bash
    git remote add origin https://github.com/yourusername/your-repo-name.git
    ```
4.  Завантажте файли на GitHub:
    ```bash
    git push -u origin main
    ```

### 3. Налаштування GitHub Pages

1.  Перейдіть у налаштування вашого репозиторію на GitHub (**Settings**).
2.  У бічному меню виберіть **Pages**.
3.  У розділі **Source** виберіть гілку **`main`** (або `master`, якщо ви її використовували) та папку **`(root)`**.
4.  Натисніть **Save**.

Через кілька хвилин ваш блог буде опубліковано за адресою, вказаною в налаштуваннях Pages.

## Як писати нові пости

1.  Створіть новий файл у папці `_posts/`.
2.  Назва файлу повинна мати формат: **`РІК-МІСЯЦЬ-ДЕНЬ-назва-посту.md`** (наприклад, `2025-11-01-my-new-post.md`).
3.  Використовуйте Markdown для написання тексту.
4.  Кожен пост повинен починатися з **Front Matter** (метаданих) у форматі YAML:

    ```yaml
    ---
    layout: post
    title: "Заголовок вашого посту"
    subtitle: "Короткий опис або підзаголовок"
    date: 2025-11-01 10:00:00 +0200
    background: 
    ---
    ```
